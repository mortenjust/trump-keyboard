    * 39 / 320 ~= 0.122
* 43 / 216 ~= 0.199 (6 P)
    * 43 / 375 ~= 0.115
* 46 / 226 ~= 0.204 (6+ P)
    * 46 / 414 ~= 0.111

* landscape always the same: scale by height
* portrait: scale linearly by width

    320 x + c = 39
    375 x + c = 43
    39 - 320 x = 43 - 375 x
    375 x - 320 x = 43 - 39
    
    **x = 4 / 55 ~= 0.072**
    
    **c = 39 - 320 x ~= 15.727**
    
    **other idea: row gaps, similarly tied to width**
